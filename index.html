Comandos basicos do git

git init

git status "é utilizado para verificar o estado do arquivo no git"

git add . (ou) git ./ "é utilizado para adicionar o codigo"

( git commit -a -m "comentario" )"é utilizado para quando realizar alteração no codigo"

Acesse seu git hub e crie um repositório

git remote add origin https://github.com/woslon/tutorial-git.git

comando para adicionar ao repositório remoto

git branch -M main

comando para tornar nossa branch principal

git push -u origin main

comando para enviar nosso repositório atual para o repositório remoto

git checkout -b "nova branch"

"é usado para uma nova branch"
